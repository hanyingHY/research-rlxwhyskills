# Focused Strengthening

## Objective

Let the user choose exactly what gets reinforced before the lite workflow broadens or escalates.

## Strengthening targets

Allow the user to focus on one or more of:

1. directions
2. datasets
3. papers
4. claim bundles

## Rule

If the user chooses strengthen mode, do not spread effort across the whole corpus by default.

Record the chosen strengthening targets first, then keep the strengthening pass aligned to those targets.
