# Lite Checkpoint Prompt Strata

## Objective

Keep the lite checkpoint prompt explicit enough that the operator can send it directly to the user.

## Minimum content

The lite prompt should contain:

1. purpose
2. recommended default
3. option impact
4. one copy-paste prompt block

## Rule

The lite skill does not need the full user, coordinator, and broadcast split of the full skill, but it must still expose a visible user-facing prompt rather than relying on implicit handoff.
