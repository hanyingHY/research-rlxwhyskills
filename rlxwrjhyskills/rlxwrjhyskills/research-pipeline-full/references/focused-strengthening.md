# Focused Strengthening

## Objective

Let the user choose exactly what gets reinforced before the program broadens or deepens.

## Strengthening targets

Allow the user to focus on one or more of:

1. directions
2. datasets
3. papers
4. claim bundles

## Rule

When the user chooses strengthen mode, do not scatter effort across the whole corpus.

Ask what should be strengthened first, record that choice explicitly, and keep work aligned to those targets until the strengthening pass is complete.

## Required control surfaces

1. `FOCUSED_STRENGTHENING_SELECTION.md`
2. `FOCUSED_STRENGTHENING_PLAN.csv`

## Minimum plan fields

1. `focus_type`
2. `focus_name`
3. `strengthen_goal`
4. `evidence_gap`
5. `preferred_action`
6. `priority`
7. `notes`
